# Handl Landing Page

Landing page oficial de **Handl** — recepcionista virtual con IA para WhatsApp que automatiza la atención al cliente, califica leads y agenda citas en Google Calendar.

Construida con [Astro](https://astro.build) y [Tailwind CSS](https://tailwindcss.com).

## Características

- **Atención 24/7** — Tu negocio nunca cierra, el recepcionista IA responde a cualquier hora
- **Calificación de leads** — Identifica automáticamente a los clientes con mayor potencial
- **Agendamiento en Google Calendar** — Citas automáticas con enlace de Google Meet incluido
- **Cumplimiento Meta 2026** — Diseñado bajo las normativas oficiales de WhatsApp Business API
- **Onboarding en minutos** — Configuración sencilla sin equipo técnico
- **Escalabilidad sin límites** — Atiende cientos de conversaciones simultáneas

## Desarrollo

| Comando                   | Acción                                        |
| :------------------------ | :-------------------------------------------- |
| `npm install`             | Instalar dependencias                         |
| `npm run dev`             | Iniciar servidor local en `localhost:4321`    |
| `npm run build`           | Construir sitio producción en `./dist/`       |
| `npm run preview`         | Vista previa del build local                  |

## Despliegue

El sitio se despliega desde la rama `main`.
